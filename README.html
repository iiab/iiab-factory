Internet-in-a-Box (IIAB) 8.0 pre-releases are thoroughly tested, and can be installed from this page! Please read our DRAFT IIAB 8.0 Release Notes.

To install IIAB 8.0 onto Raspberry Pi OS, Ubuntu 20.04+, Linux Mint 20 or Debian 11+, simply run this 1-line installer:

curl d.iiab.io/install.txt | sudo bash

WARNING: THE NOOBS OS IS *NOT* SUPPORTED, as its partitioning is very different. On a Raspberry Pi, WE RECOMMEND YOU INSTALL THE LATEST RASPBERRY PI OS, using the detailed instructions if necessary. To attempt an IIAB install onto a non-supported Linux distribution (AT YOUR OWN RISK) see also the manual/legacy instructions.

An Ethernet cable is HIGHLY RECOMMENDED during installation, as this is more reliable than Wi-Fi (and faster too!) WARNING: IF YOU CONNECT YOUR IIAB'S INTERNAL WIFI TO THE INTERNET OVER 5 GHz, YOU'LL PREVENT OLDER LAPTOPS/PHONES/TABLETS (WHICH REQUIRE 2.4 GHz) FROM CONNECTING TO YOUR IIAB'S INTERNAL HOTSPOT. See: "wifi_up_down: True" in /etc/iiab/local_vars.yml
